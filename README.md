# RAG (Retrieval-Augmented Generation) Project

A comprehensive Retrieval-Augmented Generation system that combines document retrieval with LLM-powered question answering.

## Features

- **Document Processing**: Support for PDF, TXT, and DOCX files
- **Vector Database**: ChromaDB integration for efficient similarity search
- **LLM Integration**: Google Gemini Pro for intelligent responses
- **Web Interface**: Streamlit-based user interface
- **Advanced Retrieval**: Hybrid search with similarity and keyword matching
